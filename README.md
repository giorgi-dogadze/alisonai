# React TypeScript Template

This repository is a template for starting **React + TypeScript** projects with a fully configured development environment. It comes pre-equipped with **Tailwind CSS**, **ShadCN UI**, **ESLint**, and **Prettier**, making it ready for rapid development with consistent code quality.

## Features

- **React + TypeScript**: Modern type-safe frontend development.
- **Tailwind CSS**: Utility-first CSS framework for fast UI styling.
- **ShadCN UI**: Prebuilt UI components integrated and ready to use.
- **ESLint & Prettier**: Fully configured for code quality and formatting.
- **Pre-configured rules**: Enforces consistent code style and best practices.

## Requirements

- **Node.js >= 20.19**  
  Make sure your environment meets this version to avoid issues during installation.
